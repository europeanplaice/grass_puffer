Privacy Policy

Last updated: April 29, 2026

1. Overview

Grass Puffer Diary is a browser-only web app that stores diary entries in your own Google Drive. It has no server and does not transmit your data anywhere outside of Google's infrastructure.

2. Information We Access

The only information this app accesses via OAuth is:

  • A Google OAuth access token (used solely to make Drive API calls on your behalf)
  • Files in your Google Drive that were created by this app (JSON files inside the GrassPuffer Diary/ folder)

The access token is held in browser memory only — it is never written to localStorage, cookies, or any server. It is discarded when you close the page.

3. How We Use Your Information

The access token is used exclusively to create, read, update, and delete your diary entries in Google Drive. It is not used for any other purpose.

4. Data Sharing

We do not share, sell, or transmit your data to any third party. All data remains within your own Google Drive account.

5. OAuth Scope

This app requests only the https://www.googleapis.com/auth/drive.file scope. This scope grants access only to files created by this app — it does not provide access to the rest of your Google Drive.

6. Analytics & Advertising

This app uses no analytics, advertising, or tracking of any kind.

7. Deleting Your Data

All data created by this app is stored in the GrassPuffer Diary/ folder in your Google Drive. You can delete it at any time by removing that folder. You can also revoke this app's access at any time via Google Account → Apps & services.

8. Security

Data storage and protection rely on Google Drive's security infrastructure. Because this app has no server, there is no server-side data to be breached.
